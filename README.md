# mavenapiSpringboot
Projeto construido para armazenar dados pessoais de uma pessoa

# Começando
Para executar o projeto, sera necessario instalar os seguintes programas:
- JDK 8:Necessario para executar o projeto Java
- Maven 4.0.0: Necessario para realizar o build do projeto Java
- EclipseIDE: Para desenvolvimento do projeto

# Desenvolvimento
Para iniciar o desenvolvimento e necessario clonar o projeto do Github num diretorio de sua preferencia:

cd "diretorio de sua preferencia"

git clone https://github.com/2020nani/apiSpringboot

# Construçao
Para construir o projeto com o Maven, executar o comando abaixo:

mvn clean install

# Feature
O projeto inicia a parte de api para abastecer a abertura de conta em um banco digital

# Configuração
Para executar o projeto e necessario utilizar o Eclipse, para que o mesmo identifique as dependencias necessarias para a execução. Uma vez importado para o Eclipse o projeto,sera criado um arquivo .classpath
 que ira informar qual a classe principal para a execução
# Testes
Para rodar os testes utilize o JUnit4 do Eclipse(Observação:Por padrão algumas versoes do Eclipse utiliza o J5Unit podendo dar erro, se acontecer entrar em Runconfiguration e selecionar Junit4 onde esta marcado Junit5)
